# Product Overview

statuspage-go is a Go client library for accessing the Statuspage REST API v1. It provides a clean, idiomatic Go interface for interacting with Statuspage services, allowing developers to programmatically manage status pages, components, and related resources.

Built with Go 1.25.5, the library leverages modern Go features for enhanced performance, security, and reliability while maintaining full backward compatibility.

## Key Features

- Full API coverage for Statuspage REST API v1
- Type-safe Go structs for all API resources
- Context-aware HTTP client with proper error handling
- OAuth authentication support
- JSON marshaling/unmarshaling with proper field mapping
- Modern Go 1.25.5 runtime for enhanced performance and security
- Comprehensive test coverage with property-based testing
- Full backward compatibility guarantee

## Primary Use Cases

- Automated status page management
- Component status monitoring and updates
- Integration with monitoring systems
- Custom status page dashboards and tooling

## API Documentation

Official Statuspage API documentation: https://developer.statuspage.io

## Version Requirements

- **Minimum Go version**: 1.25.5
- **Recommended Go version**: Latest stable release
- **Compatibility**: Maintains full backward compatibility with existing client code
